Manage your mesh

BRadmin is web-based administration system for 6LoWPAN border-routers. BRadmin is an easy to use automatic 6LoWPAN router as well as provide tools for commissioning and diagnostics for your 6LoWPAN deployments



Get the Code

What is a BR? A BR is a router that connects your 6LowPAN network to the Internet. Traffic is routed to and from the 6LoWPAN network automatically by the BR.

BRs can connect a variety of physical network types to a 6LoWPAN network. Commonly used networks are:

  • Ethernet
  • WiFi
  • Cellular

BRadmin requires hardware running Linux and an 802.15.4 radio.

See supported hardware for details.

Features


RPL Status

Live reporting and interrogation of the routing (RPL) in your 6LoWPAN network.

IPv6 Setup

BRAdmin makes it easy to get a globally routeable subnet in a variety of network environments: even from IPv4 networks behind NATs. Setup methods include:
  • TSP
    • Lowpan.com
    • Gogo6
  • More on the way...
    • IPv6 autoconfiguration
    • 6in4
    • NAT64

Cloud Services

BRadmin can help "glue" the software running on your low power wireless network to various Cloud Services.
